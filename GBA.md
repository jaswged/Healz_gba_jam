# GBA

A 2.9 inch screen with a 240x160 pixel resolution.
8-bit raw audio

256 sprites which can be from 8x8 to 64x64 pixels in size
Background tiles, 8x8 pixel tiles are used in the background layers if they are in tile mode.

- **A**: X
- **B**: Z
- **L**: A
- **R**: S
- **Start**: Enter
- **Select**: Backspace
